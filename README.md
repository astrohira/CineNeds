# Sessao-de-Filmes-Discord
O CineNeds é o centro de comando definitivo para comunidades que levam a sério suas noites de cinema e calls no Discord. Chega de perder links em chats infinitos ou esquecer qual filme foi combinado para o final de semana.  Com uma interface Dark Mode inspirada no próprio ecossistema do Discord, o CineNeds une praticidade e estética.
